# Strapi

### Permissions configuration

For the **Public** role:
- Application
  - Resaurant
    - find
    - findone
  - Dish
    - find

For the **Authenticated** role:
- Application
  - Resaurant
    - find
    - findone
  - Dish
    - find
  - Order
    - create
